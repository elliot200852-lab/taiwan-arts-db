## 一、關鍵查證方法

- **官方頻道地圖**：這兩期多數歌手有官方來源，優先序 official > 其餘。國語老歌靠三類官方頻道解決——① 各歌手的「XX - Topic」自動生成官方頻道（鄧麗君、姚蘇蓉、崔苔菁、余天、楊燕、冉肖玲、蘇芮、江蕙、沈文程、費玉清、蔡琴、葉啟田、陳小雲、鳳飛飛、黃鶯鶯、小虎隊等）；② 唱片公司官方頻道（麗歌唱片 Life Records Chinese、滾石 ROCK RECORDS、可登唱片 Coden Records）；③ 走轉型期的滾石／飛碟系官方 MV。
- **不收的來源**：個人上傳（shichwan、Alan Kayangan 等）一律不收；身分不明的 distributor（Timeless Music、金曲年代 GOLDEN ERA、好歌大家唱）依試點批 78rpm 前例從嚴，一律不當合格官方來源（因此蔡琴〈最後一夜〉改用 Tsai Chin - Topic 而非 Timeless；王傑、蔡藍欽因僅 Timeless 而未收）。
- **年份查證救回若干誤植**：歐陽菲菲〈嘿嘿！Taxi〉維基載明 1977 年在台發行（出 window）→ 捨；余天〈榕樹下〉為 1977 年成名曲、且改編自日本〈北國之春〉→ 捨（改收 1973 年〈又是黃昏〉）；沈文程〈來去台東〉為 1994 年自製發行→ 捨（改收 1982 年〈心事誰人知〉）。

## 二、選歌理由與版本（era-mandarin，25 首）

原唱／翻唱：**原唱 22、著名翻唱 3**（月亮代表我的心、在水一方、何日君再來——原唱陳芬蘭／江蕾／周璇無合格官方來源，改收鄧麗君官方 Topic 翻唱並於 note 交代）。source_type 全部 official。

- 意難忘（1963）｜群星會招牌歌、慎芝把日本旋律填成國語詞、美黛原唱，開場定調「電視年代」；有維基專頁載明年份與填詞。
- 一見你就笑（1967）｜鄧麗君歌廳走唱時期招牌歌，帶出她的起點（與 field-music 的海峽切角區隔）。
- 苦酒滿杯（1967）｜謝雷成名曲、慎芝改編臺語《悲戀的酒杯》；警總以「臺灣人民苦酒滿杯」查禁、換名續售半年破百萬，禁歌荒謬史一級教材。
- 負心的人（1967）｜「盈淚歌后」姚蘇蓉成名曲、與左宏元合作的姚派唱法。
- 淚的小花 / 尋夢園（1969）｜青山麗歌時期成名曲、與范婉曲的螢幕情侶組合；年份為概值。
- 晶晶（1969）｜中視開播首檔連續劇主題曲、鄧麗君早期代表作，維基專段可證。
- 今天不回家（1969）｜電影同名主題曲、左宏元作曲、紅遍東南亞；姚蘇蓉「禁唱歌后」的代表案例。
- 像霧又像花（1969）｜姚蘇蓉／左宏元姚派名曲；年份為概值。
- 往事只能回味（1970）｜劉家昌譜曲、林煌坤填詞、尤雅一炮而紅（維基專段可證），玉女歌星時代之聲。
- 祝你幸福（1972）｜鳳飛飛加盟海山首張專輯（維基載 1972/3），「帽子歌后」起點。
- 我怎能離開你 / 千言萬語（1973）｜瓊瑤電影《彩雲飛》歌曲、左宏元作曲、鄧麗君主唱，涵蓋「瓊瑤電影歌」要項。
- 心有千千結（1973）｜瓊瑤同名電影主題曲、蔡榮吉作曲、尤雅演唱（維基尤雅頁可證），補足瓊瑤電影歌女聲。
- 月亮代表我的心（1973）｜陳芬蘭 1973/5 最先錄唱、鄧麗君 1977 唱紅（維基專頁），華語金曲；收鄧麗君官方翻唱。
- 愛神（1973）｜台視綜藝天后崔苔菁，見證電視造星年代；年份為概值。
- 諾言（1975）｜「偶像始祖」劉文正出道曲、劉家昌打造。
- 在水一方（1975）｜瓊瑤電影主題曲、林家慶作曲、江蕾原唱；收鄧麗君著名翻唱。
- 何日君再來（1937）｜跨時代禁歌，國民政府疑「君」諧音「軍」而禁、對岸亦禁、劉雪庵因此被劃右派——海峽兩邊都怕的一首情歌，禁歌史頂級教材；收鄧麗君官方翻唱，年份記原唱 1937 並於 note 交代其戒嚴年代的流傳。
- 蘋果花（1968，概）｜「蘋果花歌后」楊燕招牌曲、日曲改編。
- 神秘女郎（1969，概）｜冉肖玲國語小調。
- 又是黃昏（1973，概）｜「寶島歌王」余天群星會出身後名曲。
- 鳳陽花鼓（1967）｜鄧麗君首張專輯同名曲、傳統民謠改編，她的錄音起點。
- 情難守（1970，概）｜青山麗歌時期抒情作。
- 蔓莉（1969，概）｜謝雷懷念小調。

## 三、選歌理由與版本（era-turning，31 首）

原唱／翻唱：**全部 31 首原唱、source_type 全部 official**。台語 6 首（你著忍耐、惜別的海岸、心事誰人知、一支小雨傘、愛情的騙子我問你、愛拚才會贏），其餘國語。歌手分佈：羅大佑 8、蘇芮 4、潘越雲 3、李宗盛 2、江蕙 2、蔡琴 2，加費玉清、鄭怡、沈文程、洪榮宏、陳小雲、葉啟田、鳳飛飛、黃鶯鶯、群星（明天會更好）、小虎隊等。

- 鹿港小鎮 / 之乎者也 / 光陰的故事 / 童年 / 戀曲1980（1982）｜羅大佑首張《之乎者也》，把批判寫進流行歌、敲響民歌喪鐘，滾石官方 MV 與 Lo Ta-yu 官方 Topic 皆備。
- 亞細亞的孤兒 / 未來的主人翁（1983）｜借吳濁流小說、嗩吶伴奏、美中斷交、文工會關切；羅大佑批判高峰。
- 愛人同志（1988）｜羅大佑赴港後最成功專輯同名曲、電影《棋王》主題曲。
- 一樣的月光 / 是否 / 酒矸倘賣無 / 請跟我來（1983）｜電影《搭錯車》四曲、蘇芮幕後配唱「黑色蘇芮」、臺灣首度以 MV 方式製作專輯，轉型期的核心教材。
- 天天天藍 / 守著陽光守著你 / 野百合也有春天（1982–83）｜潘越雲，滾石早期招牌與羅大佑作品。
- 生命中的精靈 / 寂寞難耐（1986）｜李宗盛從幕後走到台前的首張個人專輯。
- 心情（1983）｜鄭怡演唱、李宗盛製作，李宗盛製作生涯的重要一步。
- 夢駝鈴（1982）｜費玉清清亮嗓音，小軒／譚健常詞曲。
- 最後一夜 / 讀你（1984／1988）｜蔡琴，慎芝／陳志遠與梁弘志兩代名家詞曲，絲絨嗓音代表作。
- 你著忍耐（1983）/ 惜別的海岸（1984）｜江蕙出道期，戒嚴末期臺語歌「歌紅人不紅」的見證（維基江蕙頁明載）。
- 心事誰人知（1982）｜沈文程夜市破百萬、臺語歌壇「國歌」，蔡振南詞曲（維基沈文程頁明載）。
- 一支小雨傘（1982）｜洪榮宏十九歲成名曲，臺語歌地位確立的先聲。
- 愛情的騙子我問你（1986，概）｜陳小雲臺語舞曲名作。
- 愛拚才會贏（1988）｜送新聞局審查因「拚」被視為暴力詞遭擋、解嚴取消送審才問世（維基專頁明載），完美接到「解嚴與新臺語歌門口」。
- 掌聲響起（1986）｜鳳飛飛，依派工「發表年所在期」歸 turning。
- 明天會更好（1985）｜60 位歌手合唱、羅大佑作曲、受 We Are the World 啟發、原被規劃作國民黨選舉造勢、羅大佑「被騙」——歌替社會發聲卻被權力借用的最佳案例（維基專頁明載）。
- 只有分離（1982）｜黃鶯鶯加盟寶麗金專輯主打、蟬聯排行冠軍三月（維基黃鶯鶯頁明載）。
- 青蘋果樂園（1989）｜小虎隊出道曲、改編日本少年隊，帶進青春偶像團體年代。

## 五、給指揮官的規格疑義（請拍板）

1. **overlap 期的重複登記**：era-turning（1982–1989）與 era-new-taiwanese（1987–1999）在 1987–1989 重疊，本批一開始收了王傑〈一場遊戲一場夢〉與陳淑樺〈夢醒時分〉，`check_songs` 報 id 與 new-taiwanese 重複。已依「歌不重複登記」讓渡並補位。建議 SPEC 對 overlap 年段訂一條「先交件者優先／或指揮官指派歸屬」的裁決規則，避免各批各自登記同曲。愛人同志（1988）、掌聲響起（1986）、愛拚才會贏（1988）等未撞車，維持登記於 turning。
2. **年份概值**：部分老歌（淚的小花、尋夢園、蘋果花、神秘女郎、又是黃昏、愛神、情難守、蔓莉、愛情的騙子我問你）無單曲維基專頁，年份依歌手活躍期估列並於 note 標「概值」、sources 指向歌手維基頁。若日後補得二源再校正。此為刻意保守做法，請確認可接受。
3. **鄧麗君翻唱代原唱**：月亮代表我的心（陳芬蘭原唱）、在水一方（江蕾原唱）、何日君再來（周璇原唱）三曲，原唱皆無合格官方頻道，改收鄧麗君官方 Topic 翻唱（version: famous_cover），已於 note 交代——同 78rpm 桃花泣血記前例。請確認一致可接受。
4. **credits 從嚴**：凡查不到可靠詞曲者一律從缺（如守著陽光守著你、心情、愛情的騙子我問你、蘋果花等只列 original_singer）。〈一樣的月光〉作曲在李壽全／羅大佑間各說不一，故只列作詞（吳念真、羅大佑）＋原唱，作曲從缺。
