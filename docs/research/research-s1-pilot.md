# 臺灣歌曲線 — S1 試點研究筆記（era-78rpm 曲盤時代／era-folk 唱自己的歌）

> Writer｜2026-07-18｜歌曲線試點批（曲盤時代 order 2、民歌運動 order 5 兩期共用一檔）。
> 依 docs/SONGS-SPEC.md 逐條執行。每首入簿列選歌理由；記查證過程；末段為放棄清單與規格疑義。
> **連結全部經真實搜尋＋真實驗證**：YouTube 逐條 `youtube.com/oembed` 取回 title/author_name 核對頻道；臺史博典藏頁逐條 curl 確認 HTTP 200，並讀 WP REST（`nmth_new_item`）逐首核對詞曲／發行年／唱片編號。

## 一、關鍵查證方法（建議全線早期各期沿用）

**曲盤時代原唱難題**：1930 年代純純等原始曲盤在 YouTube 上幾乎只剩「個人帳號上傳」，依 §5 一律不可收。合格的官方翻唱頻道（滾石、華納、鄧麗君 Topic）也多半沒有這些日治老歌。

**解法＝臺史博「臺灣音聲一百年」典藏（audio.nmth.gov.tw）**。此站 2025 年改版為 WordPress，暴露 REST API：
- 搜尋：`/wp-json/wp/v2/search?search=<歌名>&subtype=nmth_new_item` → 回傳每筆典藏（單一錄音）的 id、標題、永久連結。
- 單筆：`/wp-json/wp/v2/nmth_new_item/<id>` → `link` 欄即典藏單頁永久網址；頁面含播放器＋「創作者／發行編號／專輯年份／內容」欄位，可據以核對是否為原唱古倫美亞／勝利／泰平版。
- 本批 78rpm 的 20 條 listen 有 19 條走此典藏單頁（source_type: archive），逐條 curl 回 200。**這是 roots／postwar 兩期也該沿用的取材法**（那兩期同樣卡在原唱只剩個人上傳）。

**民歌運動**則相反：滾石唱片持有新格「金韻獎」全系列版權，官方頻道「滾石唱片 ROCK RECORDS」「新格唱片 RockRecords」「滾石愛情故事 In Love」及各歌手 Topic 頻道，把幾乎整代校園民歌都上架為 Official Lyric Video。25 首全部走 official 頻道原唱。

## 二、選歌理由與版本（era-78rpm，20 首）

原唱／翻唱：**原唱 18、翻唱 2**（桃花泣血記＝典藏許石 1975 復刻；美麗島不在本期）。全部 source_type: archive（典藏單頁）；望春風、雨夜花另並列鄧麗君／方季惟官方翻唱（official）作 listen[1..2]。

- 桃花泣血記（1932）｜臺灣第一首流行歌、電影宣傳曲的起源軼事，教學價值最高。原唱純純曲盤只存個人上傳→改收臺史博典藏之許石 1975 復刻（archive／famous_cover），note 交代。
- 跳舞時代（1933）｜都會摩登／自由戀愛主題，2003 郭珍弟紀錄片片名所本；原唱古倫美亞 80273。
- 紅鶯之鳴（1933）｜與跳舞時代同片 A 面，補足摩登主題；作曲史料未定故 credits 從缺曲。
- 珈琲！珈琲！（1933）｜咖啡店女給題材，見證都會情調；與月夜愁同片。
- 老青春（1933）｜醫師兼作詞家林清月填詞，顯示流行歌創作者的多元出身。
- 望春風（1933）｜鄧雨賢代表作、最常被翻唱；典藏頁明載 1933/9 錄音、編號 80283。並列鄧麗君 Topic 官方翻唱。
- 月夜愁（1933）｜四月望雨之一；戰時被改成〈軍夫の妻〉的來源曲。
- 雨夜花（1934）｜日治至戰後最傳唱悲歌；並列方季惟（華納官方）＋鄧麗君（Topic）兩條翻唱，示範同曲跨時代兩種聲音。
- 街頭的流浪（1934）｜臺灣第一首被查禁的創作流行歌（失業／左翼題材），禁歌史核心教材；泰平 82003。
- 對花（1935）｜男女對唱、四季花名互答，題材活潑；利家紅標 T1007。
- 春宵吟（1935）｜與對花同片 B 面，鄧雨賢抒情小品。
- 碎心花（1935）｜周添旺／鄧雨賢閨怨小曲；古倫美亞 80330。
- 心酸酸（1936）｜勝利唱片暢銷悲歌、秀鑾主唱，代表古倫美亞以外的勝利系；F1076。
- 桃花鄉（1936）｜改編自 1928 上海〈桃花江〉，見證兩岸流行歌交流；勝利 F1075。
- 白牡丹（1936）｜取材藝旦曲改寫為流行小曲；典藏題「白牡丹（流行小曲）」，與同名北管曲區辨（note 標明）。
- 四季紅（1938）｜四月望雨中最輕快者；戰後因「紅」字改名〈四季謠〉的軼事；日東 A101。
- 誉れの軍夫（1938）｜雨夜花旋律改填日語從軍詞，「歌被時代改寫」的一級教材；日本コロムビア 29704。
- 軍夫の妻（1938）｜月夜愁旋律改填，與上曲同片。
- 滿面春風（1939）｜鄧雨賢戰前晚期作品；利家紅標 T1161。
- 春宵夢（1939）｜周添旺自作詞曲，顯示作詞家跨足作曲。

## 三、選歌理由與版本（era-folk，25 首）

原唱／翻唱：**原唱 25、翻唱 0**（含福爾摩沙合唱團之美麗島 1 條記為 famous_cover，見疑義）。source_type：official 24、other 1（美麗島）。

- 如果 / 再別康橋（1977）｜金韻獎第一輯，運動由校園走向唱片市場的起點；再別康橋把徐志摩新詩唱進校園。
- 蘭花草（1978）｜胡適白話詩〈希望〉入歌，最普及的傳唱教材。
- 看我！聽我！／媽媽說的故事（1978）｜金韻獎歌手包美聖代表曲（credits 僅列原唱，避免臆測詞曲）。
- 讓我們看雲去（1978）｜陳明韶名曲，唱年輕人的嚮往。
- 橄欖樹／歡顏／答案／走在雨中（1979）｜李泰祥×齊豫路線，把新詩與藝術歌曲帶進民歌；歡顏亦為同名（以民歌手為主角）電影主題曲。
- 秋蟬（1979）｜東吳雙人重唱奪金韻獎的清亮和聲。
- 捉泥鰍（1979）｜侯德健童趣之作，兒歌式民歌。
- 木棉道／阿美阿美／雨中即景（1979）｜王夢麟三首，涵蓋抒情、俏皮、幽默三面。
- 美麗島（1979）｜李雙澤曲、禁歌史與「美麗島」一詞的政治後續，教學價值極高（連結見疑義）。
- 歸人沙城／拜訪春天（1980）｜排灣族歌手施孝榮，帶入原住民歌手在民歌時代的身影。
- 生命的陽光（1980）｜木吉他合唱團，李宗盛樂壇起點的小故事。
- 龍的傳人／曠野寄情（1980）｜中美斷交年頭的集體記憶；李建復。
- 忘了我是誰（1980）｜李敖獄中詩入歌，來歷特殊。
- 微風往事（1981）｜鄭怡；金韻獎第八輯。
- 廟會（1981）｜把民間廟會寫進民歌的難得鄉土題材；作詞賴西安即李潼。
- 就在今夜（1982）｜丘丘合唱團，樂團形式替民歌接上搖滾年代，收束到下一期（黑色狂潮）門口。

## 五、給指揮官的規格疑義（請拍板）

1. **典藏單頁作 listen[0] 與 check_songs 驗證方式**：78rpm 主連結多為 audio.nmth.gov.tw 典藏單頁（非 YouTube）。§5／§6 明文允許「典藏單頁」且對非 YouTube 走 HTTP 200 驗證——本批已逐條 curl 200。請確認 check_songs.py 對 `audio.nmth.gov.tw/audio/...` 這類 URL 走 200 檢查（含 UTF-8 percent-encoded 路徑），且首頁歌名自動掛鏈導向典藏頁符合「歌名即連結」精神。**強烈建議把此 WP REST 取材法寫進 SPEC，供 roots／postwar 沿用。**
2. **美麗島的來源分級**：唯一合格連結為福爾摩沙合唱團官方合唱版（source_type: other、version: famous_cover）；楊祖珺 1977/1979 原唱在合格典藏頻道未見。已收並於 note／banned 交代原委。此曲教學價值極高（禁歌史＋事件命名），但屬本批唯一 other 來源——請確認可留，或改為純敘事帶過、退出登記簿。
3. **桃花泣血記以復刻代原唱**：以臺史博典藏之許石 1975 復刻（archive／famous_cover）代替只存個人上傳的 1932 純純原唱。請確認此處理可接受（同理未來 roots 期恆春調等亦可能如此）。
4. **credits 從嚴**：凡典藏頁未明載演唱者／作曲者者一律從缺不臆測（如紅鶯之鳴曲、看我！聽我！詞曲、歸人沙城詞曲）。此為刻意保守，若日後補得可靠二源再增補。
