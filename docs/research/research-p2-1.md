## 一、美術（field-art）

### 貫串線索（一句話）
一群人如何在日治到戰後這半世紀間，把「臺灣長什麼樣子」第一次留在畫布與石頭上——舞台從東京帝展搭到家門口的台展，題材都是腳下的土地，路徑在廟宇工藝與學院美術之間來回，終點是連規矩都不甩的素人洪通。

### 段落分工
1. **舞台搭起來了：帝展與台展** — 制度線索。帝展（東京最高殿堂）→ 黃土水首位進東京美校＋1920《山童吹笛》入帝展 → 陳澄波 1926 首以油畫入帝展 → 1927 台展在臺灣本地開辦（十回，1927–1936）、府展續至 1943 → 台展三少年（陳進、郭雪湖、林玉山）、陳進唯一女性且後入帝展。
2. **畫的都是腳下的地方** — 地方線索（橫看整座島）。陳澄波畫嘉義／淡水、郭雪湖畫大稻埕、陳進畫新竹香山仕女。串 geo-db：chiayi-city / new-taipei / taipei / hsinchu-city。
3. **從廟裡出發，又走回廟裡** — 工藝×學院線索。黃土水（艋舺木匠鋪）、朱銘（通霄慈惠宮廟宇木雕）都從廟裡練手；李梅樹反向把學院帶回三峽祖師廟（百鳥朝梅寫生、請郭雪湖／陳進畫底稿）。串 taipei / miaoli / new-taipei。
4. **舞台之外的人** — 邊界線索收束。洪通（臺南北門、五十歲、無師自通、1972 南鯤鯓廟牆一夕成名）。收在「創作不需要許可證」，回扣黃土水與洪通兩端。

## 二、音樂（field-music）

### 貫串線索（一句話）
臺灣的歌一頭連著泥土（月琴／民謠／口傳）、一頭連著五線譜（學院／唱片／國際舞台），四位歌者從恆春走唱唱到飛過海峽，這條路就是這座島自己走過的路。

### 段落分工
1. **從泥土裡長出來的歌** — 口傳民謠。陳達（恆春月琴、《思想起》、1967 民歌採集運動被史惟亮／許常惠尋獲、1971 唱片）。串 pingtung。
2. **五線譜遇上台語歌** — 唱片工業。鄧雨賢（龍潭客家、師範學西樂、1933 古倫美亞、四月望雨、西洋和聲配台語聲調→旋律可反覆翻唱）。串 taoyuan。
3. **讓世界聽見臺灣** — 國際舞台。江文也（臺北出生原籍三芝、1934 返鄉見水田白鷺鷥、《臺灣舞曲》1936 柏林奧運藝術競賽獲獎＝臺灣人第一面奧運獎牌）。串 taipei。
4. **越過海峽的聲音** — 冷戰邊界。鄧麗君（雲林褒忠、禁歌《何日君再來》、1979《甜蜜蜜》破百萬、日本三連霸、金門勞軍）。串 yunlin、kinmen。收在回扣恆春月琴（首尾呼應）。

## 三、文學（field-literature）

### 貫串線索（一句話）
日治到戰後，臺灣作家第一次用貼近自己講話的白話文替土地與沉默的人寫字；三人共通的兩條明線——「一手撐著艱難生活（行醫／送報／種田）、一手寫字」，以及「殖民下各自不同的語言處境」（賴和貼台灣口語的白話／楊逵得獎的送報伕是日文／鍾理和自學魯迅老舍的中文）——收在結尾點明，作為橫看整座島的脊。

> 深化說明：初稿三段敘事僅 ~1,110 字（因可連人物僅 3 位，明顯低於 geo-db 主題頁與本批美術 2,281／音樂 1,888 字水準）。第二輪用「已在人物頁查證過的事實」補厚，未新增未查證主張：段一加〈南國哀歌〉（1931，霧社）與「刻意寫台灣口語白話」；段三加台妹大四歲／無血緣、〈原鄉人〉（1959）、自學魯迅老舍、公學校與未唸完師範；並新增結尾「語言處境」段把三人綁成一條明線。補厚後三段敘事約 1,700+ 字。新增來源僅一條需另驗活性：維基「鍾台妹」（curl 200，已加入 [^6]），其餘全沿用人物頁既有腳註。

### 段落分工（只 3 段，因可連人物僅 3 位）
1. **用自己的話，替沒有聲音的人寫字** — 白話文學開端。賴和（彰化、本島衫、除夕燒帳單、彰化媽祖、二林蔗農事件 1925→〈一桿「稱仔」〉1926）＋制度角色（1921 參與臺灣文化協會、1926 主持《臺灣民報》文藝欄、提攜後進）→ 直接牽線到楊逵。串 changhua。
2. **壓不扁的：替勞動者發聲** — 抵抗文學。楊逵（新化、〈送報伕〉原名日文〈新聞配達夫〉1932 連載／1934 入東京《文學評論》＝首位進日本中央文壇、街頭運動被捕、葉陶「官費蜜月」、1949〈和平宣言〉坐牢十二年、〈壓不扁的玫瑰〉1976 入國中課本）。串 tainan。
3. **寫土地，也寫「原鄉」** — 鄉土文學。鍾理和（美濃客家、同姓不婚私奔東北八年、「原鄉」的兩頭落空、《笠山農場》、1960 咳血逝＝「倒在血泊裡的筆耕者」）。串 kaohsiung。收在把賴和行醫／楊逵送報／鍾理和種田三人綁在「一手生活一手寫字」。

## 棄用素材與原因

- **人間福報〈雕塑大師朱銘〉（merit-times.com）**：ju-ming 人物頁的通霄／李金川來源。curl 與 WebFetch 皆回 **403**（站方擋非瀏覽器 UA，疑 WAF）。無法在本 session 依「腳註逐條確認活的」紀律驗活 → **棄用，改維基「朱銘（雕塑家）」**（已 WebFetch 逐條確認載明通霄慈惠宮、李金川、楊英風、太極系列）。
- **想想論壇〈畫畫是為了爽快！洪通〉（thinkingtaiwan.com）**：hung-tung 人物頁第二來源。DNS 解得到（139.162.116.195）但連線被拒（curl 回 **000**，connection refused，站疑暫時或長期下線）→ **棄用，改單引維基「洪通」**（南鯤鯓成名、五十歲起畫等關鍵事實齊備）。
- **「東方畢卡索／靈異畫家」綽號**：hung-tung 頁有，但綽號屬評價性、且穩妥來源正是被棄的想想論壇 → 本頁不寫綽號，只寫可由維基支撐的事實（無師自通、廟牆展出、一夕成名），避免掛空。
- **林玉山**：台展三少年之一，但不在本站現有 19 位人物頁 → 文中提名以交代「三少年」組合，但**不加連結**（避免死鏈）。
